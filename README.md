# **Everglow19000.github.io**
A website that represents FTC team Everglow #19000, especially for scouting in competitions.<br>
<br>

Feel free to fork this repository and adapt it for your own needs. For questions or help, contact 'electrocaruzo' on discord, or send an email to <fird.ftc@gmail.com>.<br>

## How to set up for yourself:
This website's scouting framework uses Firebase. Forking requires you to use your own API keys for Firebase, the FTC Events API and a proxy for it. An online tool for generating encrypted API keys in the format needed based on a custom password can [be found here](https://karpada2.github.io/encrypting.html).<br>
Create a Firebase project and follow their instructions to open and connect a Realtime Database. Then, where needed, replace the encrypted API keys present in this repository with yours using the online tool.<br>
